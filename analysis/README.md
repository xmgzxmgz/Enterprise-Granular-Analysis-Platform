# Data Analysis Module

This directory contains Python scripts for data analysis and model training.

## Prerequisites

- Python 3.8+

## Setup

1. Create a virtual environment (recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # macOS/Linux
   # venv\Scripts\activate   # Windows
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Scripts

- `shap.py`: Main analysis script using SHAP values.
- `temp_1.py`: Temporary clustering experiment.

## Data

- `training_dataset1.xlsx`: Input dataset for analysis.
